# Wata Study - Локальный запуск

Это приложение можно запустить на твоем компьютере.

## Как запустить:

1. Скачай все файлы проекта (используй иконку "Export" или "Settings -> Download ZIP" в AI Studio).
2. Убедись, что у тебя установлен **Node.js** (скачать: [nodejs.org](https://nodejs.org/)).
3. Запусти файл `run_local.bat` (для Windows).

Скрипт сам установит нужные компоненты и откроет приложение в твоем браузере.

## Использование с Ollama:

Для работы ИИ-функций:
1. Запусти Ollama на своем ПК.
2. Разреши CORS для Ollama (установи переменную окружения `OLLAMA_ORIGINS="*"`).
3. Запусти мост-скрипт (если используешь его).

По умолчанию приложение ищет ИИ по адресу: `http://192.168.1.115:8000` (ты можешь изменить это в файлах `src/components/AddTaskModal.tsx` и `src/screens/TaskDetailScreen.tsx`).
